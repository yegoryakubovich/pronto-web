
# Pronto WEB

## 1. About

Pronto is a dating app. The backend is based on Flask

## 2. Install

#### 2.1 Install requirements
```pip install -r requirements.txt```
#### 2.2 Create environment variables
- SECRET_KEY
- DB_NAME
- DB_USER
- DB_PASSWORD
- DB_HOST
- DB_PORT
#### 2.3 Run server
```python main.py```
